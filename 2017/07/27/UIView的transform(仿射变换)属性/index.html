UIView的transform(仿射变换)属性

transform 是 CGAffineTransform(结构体)类型

struct CGAffineTransform {

    CGFloat a, b, c, d;
    CGFloat tx, ty;
}
  • 对View进行一次变换(也就是修改transform属性的默认值)
  • 旋转
CGAffineTransform CGAffineTransformMakeRotaion(CGFloat angle) 

注意: angle是弧度制
  • 缩放
CGAffineTransform CGAffineTransformMakeScale(CGloat sx, CGFloat sy)
  • 平移
CGAffineTransform CGAffineTransformMakeTranslation(CGFloat tx, CGfloat ty)

  • 在某个transform的基础上进行叠加

  • 旋转

CGAffineTransform CGAffineTransformRotaion(CGAffineTransform t, CGFloat angle)
  • 缩放
CGAffineTransform CGAffineTransformScale(CGAffineTransform t, CGloat sx, CGFloat sy)
  • 平移
CGAffineTransform CGAffineTransformTranslation(CGAffineTransform t, CGFloat tx, CGfloat ty)
  • 清空之前设置的transform属性 
xxView.transform = CGAffineTransformIdentity
